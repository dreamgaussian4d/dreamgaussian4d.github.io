DreamGaussian4D: Generative 4D Gaussian Splatting

Code

Framework

DreamGaussian4D generates 4D contents in minutes by leveraging 4D Gaussian Splatting. We first obtain a static 3D GS model and a driving video and then optimize a deformation network. Finally, per-frame meshes can be refined with a video-to-video pipeline.
